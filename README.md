# Sales Performance Dashboard

## Overview

This project was completed as part of a Data Analyst Internship (Task 3). The objective was to design an interactive dashboard in Tableau that helps business stakeholders monitor sales performance, analyze trends, and support data-driven decision-making.

## Objective

Develop an interactive sales dashboard that presents key business metrics, visualizes sales performance, and enables users to explore data through filters and interactive charts.

## Dataset

- **Dataset:** E-Commerce Sales & Profit Analysis
- **Source:** Kaggle
- **Format:** CSV

## Tool Used

- Tableau Public

## Dashboard Features

- KPI Cards
  - Total Sales
  - Total Profit
  - Total Orders
- Sales by Category
- Sales Trend Analysis
- Profit by Month
- Profit by Region
- Interactive Filters
  - Region
  - Category
  - Sales

## Files Included

- `sample_ecommerce_sales_data.csv` – Dataset
- `Sales_Performance_Dashboard.twb` – Tableau workbook
- `Dashboard.pdf` – Dashboard export
- `Presentation.pptx` – Project summary
- `Insights.md` – Key business insights

## Outcome

The dashboard enables users to monitor business performance, identify trends, compare regional and category performance, and make informed business decisions through interactive visualizations.
